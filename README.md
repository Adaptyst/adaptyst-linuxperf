# adaptyst-linuxperf

[![License: GNU GPL v2 only](https://img.shields.io/badge/license-GNU%20GPL%20v2%20only-blue)]()
[![Version](https://img.shields.io/github/v/release/adaptyst/adaptyst-linuxperf?include_prereleases&label=version)](https://github.com/Adaptyst/adaptyst-linuxperf/releases)

linuxperf is an Adaptyst system module currently built on top of Linux "perf" with custom patches. It analyses comprehensively on- and off-CPU activity of single- and multi-threaded programs.

## Disclaimer
This is currently a dev version and the module is under active development. Bugs are to be expected. Use at your own risk!

All feedback is welcome.
